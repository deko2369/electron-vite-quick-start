# vite-electron-quick-start

## Usage

```bash
# Clone this repository
git clone https://github.com/yingpegnsha/vite-electron-quick-start
# Go into the repository
cd vite-electron-quick-start
# Install dependencies
yarn install
# Run the app
yarn run start
```
